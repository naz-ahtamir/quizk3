Konfigurasi Firebase

Gratis · Realtime sync · Tidak perlu server

1
Buka console.firebase.google.com → Add project (beri nama bebas)
2
Di sidebar: Build → Realtime Database → Create database → pilih region Asia → Start in test mode
3
Klik ⚙️ Project Settings → General → Your apps → Add app (Web) → copy firebaseConfig
4
Paste config JSON di bawah ini
